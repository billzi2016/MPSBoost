# MPSBoost Documentation

MPSBoost is a tree-model learning library for Apple Silicon. The public release is
`0.3.0`, with native CPU/MPS backends, sklearn-style estimators, native CPU softmax
multiclass classification, forests, advanced regression objectives, explanations,
anomaly detection, and ranking entry points.
