# Gradient Boosting

Gradient boosting uses the native tree engine. CPU is the correctness oracle; MPS
accelerates suitable histogram hot paths. Regressors support squared error, quantile,
Poisson, Tweedie, weights, constraints, regularization, growth strategies, and I/O.
Classifiers support binary logistic, native CPU multiclass softmax, explicit OvR
compatibility, probabilities, decision functions, and sklearn model selection.
