# Release 0.5.0

`mpsboost==0.5.0` is the 0.x hardening release.

```bash
python -m pip install mpsboost==0.5.0
```

## Scope

`0.5.0` keeps the `0.4.0` large-scale validation scope and adds customer-facing hardening:
portable backend policies warn and continue through native CPU compatibility when an external
runtime is not active for the current estimator, summaries record both requested and effective
backends, and known-issue status is documented before any `1.0.0` commitment.

See the [0.5.0 Release Audit](../project/RELEASE_AUDIT_0.5.0.md),
[0.5.0 Known-Issue Audit](../project/KNOWN_ISSUES_0.5.0.md), and
[0.5.0 Performance Report](../project/PERFORMANCE_0.5.0.md).
