# Release 0.3.0

`mpsboost==0.3.0` is published on PyPI.

```bash
python -m pip install mpsboost==0.3.0
```

## Scope

`0.3.0` is the v2 arboretum milestone. It includes sklearn-style classifiers,
native CPU multiclass softmax, RandomForest, ExtraTrees, DecisionTree,
CatBoost-like numeric estimators, quantile/Poisson/Tweedie objectives, feature
importance and explanations, CPU-suitable IsolationForest and ranking, and MPS
environment guidance.

See the [0.3.0 Release Audit](../project/RELEASE_AUDIT_0.3.0.md) for artifact details.
