# Release 1.0.0

`mpsboost==1.0.0` is the stable customer-commitment release for the native CPU/MPS MPSBoost scope.

```bash
python -m pip install mpsboost==1.0.0
```

## Scope

`1.0.0` freezes the public native CPU/MPS estimator scope hardened through the `0.x` line. It keeps
external SHAP, sklearn, XGBoost, and CUDA paths explicit and observable rather than hidden behind a
native backend name.

See the [1.0.0 Release Audit](../project/RELEASE_AUDIT_1.0.0.md),
[1.0.0 Known-Issue Audit](../project/KNOWN_ISSUES_1.0.0.md), and
[1.0.0 Performance Report](../project/PERFORMANCE_1.0.0.md).
