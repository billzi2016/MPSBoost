# Release 0.4.0

`mpsboost==0.4.0` is the 0.x feature release after large-scale and real-world validation work.

```bash
python -m pip install mpsboost==0.4.0
```

## Scope

`0.4.0` keeps the native CPU/MPS implementation as the default MPSBoost backend and adds the
release-facing pieces needed after the v2 arboretum milestone: optional SHAP integration
diagnostics, explicit portable-backend diagnostics, project-local opt-in real-world dataset
caches, and the S18 real-world acceptance report.

This is not a `1.0.0` stability commitment. The full real-world matrix, model-quality audit,
end-to-end performance audit, peak-memory audit, wheel/model-size audit, permission audit,
artifact hashes, and final user confirmation remain release gates before any `1.x` version.

See the [0.4.0 Release Audit](../project/RELEASE_AUDIT_0.4.0.md) and Changelog from the project
documentation for artifact and history details.
