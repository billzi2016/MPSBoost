# 0.3.0 发布

`mpsboost==0.3.0` 已发布到正式 PyPI。

```bash
python -m pip install mpsboost==0.3.0
```

## 发布范围

`0.3.0` 是 v2 arboretum milestone，包含：

- sklearn-style classifier
- native CPU multiclass softmax
- RandomForest / ExtraTrees / DecisionTree
- CatBoost-like numeric estimators
- quantile / Poisson / Tweedie
- feature importance and explanations
- CPU-suitable IsolationForest
- CPU-suitable LearningToRankRegressor
- MPS environment guidance
