# 0.5.0 发布

`mpsboost==0.5.0` 是 0.x 加固版本。

```bash
python -m pip install mpsboost==0.5.0
```

## 范围

`0.5.0` 保持 `0.4.0` 大规模验证范围，并增加客户侧加固：external runtime 没有对当前 estimator 激活时，portable backend policy 会 warning 并通过 native CPU compatibility 继续运行；summary 会记录 requested/effective backend；known-issue 状态会在任何 `1.0.0` 承诺前写入文档。

见 [0.5.0 发布审计](../project/RELEASE_AUDIT_0.5.0.md)、[0.5.0 Known-Issue 审计](../project/KNOWN_ISSUES_0.5.0.md) 和 [0.5.0 性能报告](../project/PERFORMANCE_0.5.0.md)。
