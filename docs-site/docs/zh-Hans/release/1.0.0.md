# 1.0.0 发布

`mpsboost==1.0.0` 是 native CPU/MPS MPSBoost 范围的 stable customer-commitment release。

```bash
python -m pip install mpsboost==1.0.0
```

## 范围

`1.0.0` 冻结经过 `0.x` line 加固的 public native CPU/MPS estimator 范围。External SHAP、sklearn、XGBoost 和 CUDA 路径保持显式、可观测，不隐藏在 native backend 名称后。

见 [1.0.0 发布审计](../project/RELEASE_AUDIT_1.0.0.md)、[1.0.0 Known-Issue 审计](../project/KNOWN_ISSUES_1.0.0.md) 和 [1.0.0 性能报告](../project/PERFORMANCE_1.0.0.md)。
