# 0.4.0 发布

`mpsboost==0.4.0` 是当前 0.x 功能版本。

```bash
python -m pip install mpsboost==0.4.0
```

## 发布范围

`0.4.0` 保持 native CPU/MPS 作为默认 MPSBoost 后端，并补上 v2 arboretum 里程碑之后发布所需的面向用户部分：可选 SHAP 集成诊断、显式 portable-backend 诊断、项目内 opt-in 真实世界数据集 cache，以及 S18 真实世界验收报告。

这不是 `1.0.0` 稳定承诺。完整真实世界矩阵、模型质量审计、端到端性能审计、内存峰值审计、wheel/model 体积审计、权限审计、artifact hash 和用户最终确认，仍然是任何 `1.x` 版本发布前必须关闭的发布门。

artifact 和历史细节见项目文档中的 [0.4.0 发布审计](../project/RELEASE_AUDIT_0.4.0.md) 与 Changelog。
