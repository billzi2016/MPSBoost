# MPSBoost 0.2.0rc0 发布审计

`0.2.0rc0` 是正式 `0.2.0` 之前的 cache and stability release candidate。

包含：

- versioned L2 cache key；
- atomic write 和 checksum validation；
- damaged-cache invalidation；
- public cache diagnostics、explicit cache creation 和 safe cache clearing；
- 更严格的 CI cache 边界。

不包含：

- all-tree model family；
- classification breadth；
- advanced objective 和 explanation。
